# Haotrader证券交易系统
这是一个证券交易系统的项目，为了方便部署，拆分出了多个模块。目前只提供服务端程序和接口，前端或者客户端期待更多有缘人一起来参与开发。
    
    Haotrader 证券交易撮合引擎
    Haoquote  证券交易行情模块
    Haobase   用户资产、订单记录、订单结算

Github: https://github.com/yzimhao/trading_engine
