  HaoTrader适用于各种金融证券交易场景。拥有高性能的订单撮合、实时结算、行情计算、实时推送等功能。支持数据持久化，故障重启快速恢复数据。
  
  配置灵活，允许用户根据自身需求自定义配置各模块独立运行。
